Microsoft Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.

Choose Start and then expand All Programs. Choose Visual Studio 2019 > Visual Studio Tools > Developer Command Prompt for VS 2019, or the command prompt you want to use.The “Console” Tab. The heart of any developer tool is the “console” tab where you can output debug and execute commands in to the current web page.
